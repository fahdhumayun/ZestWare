*******************************************************************************
*
*************************** Customer Module ************************************
********************************************************************************


How to run the CustomerWelcomeScreen Unit Test:


For LINUX/MAC OS (the file is in runnable jar format):

access to the director
cd 1_code/Customer/CustomerWelcomeScreen/src


java -jar CWS_UnitTest.jar

Alternatively - in Eclipse:

when the CustomerWelcomeScreen folder is opened in eclipse, the class UnitTests 
can be run and it will display the test results with the valid and invalid inputs 
and it will also show the passed tests and failed tests.
